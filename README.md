# Stocks-Watchlist

### About the Project 
Full stack web application in which users can search and monitor the prices of their favorite stocks. 
Through the implementation of protected routes and interceptors, users can securely register an account to create personalized watchlists with custom names and stock entries. 
The App utilizes the Yfinance API which connects users to real financial data via Yahoo Finance, enabling users to receive relevant up-to-date information

### Built with:
- React JS
- Django Python
- Django Rest Framework
- Yfinance API
